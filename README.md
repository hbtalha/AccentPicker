# Accent Picker

Accent Picker is a lightweight accent picker for Linux, inspired by **PowerToys Quick Accent** on Windows. It allows you to quickly insert accented characters by holding a key and selecting the desired accent from a popup, similar to the Windows experience.  

For reference, see the original PowerToys Quick Accent: [PowerToys Quick Accent](https://learn.microsoft.com/en-us/windows/powertoys/quick-accent).

## Features

- Displays a popup when holding letter keys, allowing selection of accented versions.
- Designed to closely mimic the behavior of PowerToys Quick Accent on Windows.
- Works on **X11 displays only**, as it relies on **Qt X11 Extras** for certain windowing functionality.
